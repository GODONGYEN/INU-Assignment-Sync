# Troubleshooting

## `.env`가 없어요
- `bash setup.sh`를 다시 실행하거나 `.env.example`을 `.env`로 복사하세요.

## Playwright Chromium 실행 실패
- `playwright install chromium`를 다시 실행하세요.

## Calendar 권한 오류
- `시스템 설정 > 개인정보 보호 및 보안 > 캘린더`에서 터미널 앱 권한을 허용하세요.

## 과제가 0개 수집돼요
- LMS 로그인 상태
- 월간 캘린더 페이지 접근 여부
- selector 설정

을 다시 확인하세요.
