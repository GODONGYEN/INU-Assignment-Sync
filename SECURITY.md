# Security Policy

## 지원 범위

이 프로젝트는 개인 학사 일정 보조 도구이며, 다음 원칙을 유지합니다.

- INU LMS 로그인은 사용자가 직접 수행합니다.
- CAPTCHA, SSO, 학교 보안장치 우회 기능을 구현하지 않습니다.
- 사용자 비밀번호를 코드에 저장하지 않습니다.
- 민감한 세션 정보를 로그에 남기지 않도록 주의합니다.

## 비밀 정보 처리

다음 파일은 GitHub에 올리면 안 됩니다.

- `.env`
- `data/*.sqlite3`
- `logs/*.log`

`.env.example`만 저장소에 포함하고, 실제 개인 설정은 로컬에서만 관리하세요.

## 취약점 제보

보안 문제를 발견했다면 공개 이슈에 비밀번호, 쿠키, 세션값, 학교 계정 정보를 적지 마세요.

제보 시 포함하면 좋은 정보:

- 어떤 기능에서 문제가 발생했는지
- 재현 방법
- 영향 범위
- 로그 중 민감하지 않은 부분
